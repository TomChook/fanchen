# 当前进度

## 已完成

- 主界面已稳定为左侧人物与案头、右侧主工作区的结构。
- 挂机策略与手动操作已迁入谋局台，不再与人物栏重复占位。
- 山河图与江湖纪事已改为独立浮窗。
- 浮窗已支持拖拽、关闭、最小化、基础停靠与布局记忆。
- 地图窗口已放大，地图与地脉详情的主次关系已重新分配。
- 产业系统已有田产、工坊、铺面与行会订单。
- NPC 已有年龄、阶段、职业、目标、势力流动与近年履历。
- 个人记录与阶段性统计已从常驻人物卡移入低频人物簿窗口。
- 案头里的重复“功能直达”已移除，避免和主工作区页签重复。
- 已加入“手动操作”模式，世界照常推进，但不会自动替玩家执行日常动作。

## 当前可用

- 主工作区：行囊、产业、商店、拍卖、战斗、群像、势力、天机。
- 谋局台控制：挂机策略切换、手动修炼、主动冲关、追查机缘、顺手做买卖、短暂调息。
- 低频窗口：山河图、江湖纪事、人物簿。
- 常驻高频信息：人物状态、当前焦点、最新风闻、活跃秘境、当前门路。

## 接下来优先做

- 家宅系统：买屋、扩院、仓房与工位摆放。
- 师承谱系：NPC 收徒、带徒与传承链。
- 婚配与家族：道侣、子嗣、继承与家业流转。
- 地区治安与税赋：直接影响跑商、铺面和产业收益。
- 黑市、镖局、赌场：补足低阶生计路线。
- 更完整的窗口停靠与多区布局。

## 还没做满

- 目前只有地图、纪事、人物簿进入了低频窗口体系，更多低频模块还没迁入。
- 窗口停靠还是基础版，尚未做吸附、多列分区和更完整的桌面编排。
- NPC 的婚育、家族、墓志与代际传承还未形成完整闭环。
