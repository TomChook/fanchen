# 功法、学识与誊抄方案

## 1. 目标

- 当前版本先固定为“单阶熟练度模型”，功法学会后只走熟练度到圆满这一条线。
- 满熟练必须带来明确变化，不再只是数字到顶。
- 誊抄必须可控，不能退化成“包里有书就无限复制”。
- 新增“学识札记”作为一次性学习用品，补基础属性与生活技能成长曲线。

## 2. 功法规则

### 2.1 单阶定义

| 状态 | 含义 | 玩家反馈 |
|---|---|---|
| 入门 | 学会并入门，获得基础效果 | 可装备、可施放、可开始练熟练度 |
| 圆满 | 达到该功法对应熟练度阈值 | 立刻获得圆满加成，并解锁无限誊抄资格 |

### 2.2 圆满收益

- 心法圆满偏向补修炼、突破、气机上限与宗门功能。
- 术法圆满偏向补倍率、状态层数或微量降耗。
- 圆满后熟练度不再继续叠加，不做伪成长。

### 2.3 誊抄规则

- 必须先学会该功法。
- 必须将该功法练至熟练度圆满。
- 解锁后可以无限次誊抄，但每次仍需消耗固定材料与真气。
- 誊抄出来的是实体秘籍物品，可以交易、收藏或收入藏经阁。
- 发现专属功法仍然不进入普通流通池，只能通过悟道拿到“原典来源”，之后再走誊抄产出。

## 4. 学识札记方案

### 4.1 定义

- 学识札记属于“功法秘籍”下的新子类，但不归入已学功法列表。
- 使用后直接消耗，立刻结算属性与生活技能收益。
- 已读过的学识不能再次学习。

### 4.2 设计边界

- 只给基础属性和生活技能，不给主动战斗按钮。
- 适合作为市场、商盟、工坊、藏书摊的新掉落来源。
- 用来补足非战斗成长线，避免所有高价值书册都挤向功法。

### 4.3 当前条目

| 学识 | 功能定位 |
|---|---|
| 田亩轮作札记 | 农务启蒙，兼补少量悟性 |
| 百工开炉要录 | 工艺启蒙，顺带抬一点战力 |
| 行商算筹录 | 商道启蒙，补基础魅力 |
| 门礼应对篇 | 社交向学识，适合经营和门路玩法 |
| 望气辨脉录 | 悟性向学识，兼少量战力 |
| 驿路地志抄 | 跑商与地理向学识，给多线经营留空间 |

## 5. 控产逻辑

- 誊抄材料锁定为“空白册页 + 灵墨 + 真气”。
- 册页和灵墨既能在市场流通，也能在工坊配方中生产，形成可控供给链。
- 稀有功法的誊抄成本会抬高册页与灵墨数量，阻止顶级秘籍廉价泛滥。
- 这样既保留了“圆满后可卖书、可传承”的爽感，也不会把经济系统直接打穿。

## 6. 后续可扩展方向

- 后续若要扩容，可在“圆满”之上追加心法异象、术法变式或誊抄品质系统。
- 学识札记可以继续向产业、宗门、社交三个方向扩容。
- 如果后续开放书坊产业，可直接围绕册页、灵墨、誊抄订单继续做商业线。
