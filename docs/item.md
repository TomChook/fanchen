# 物品、装备与功法总览

## 1. 分类总表

| 主类 | 说明 | 对应章节 |
|---|---|---|
| 材料与资源 | 采集、制造、经营、突破中会反复流转的基础资源与稀有素材。 | 第 3 节 |
| 装备 | 直接穿戴或持有并即时影响角色战力与生存的装备。 | 第 4 节 |
| 凭证与功能物 | 丹药、契书、牌照、信物与功能型道具。 | 第 5 节 |
| 功法秘籍 | 用于学习心法与术法的实体秘籍物品。 | 第 6 节 |

## 2. 品阶表

| rarity | 显示品阶 | 说明 |
|---|---|---|
| common | 凡品 | 市井常见、低阶修士或凡人可频繁接触。 |
| uncommon | 灵品 | 已有明显灵性，多见于学徒、外门或基础修行资源。 |
| rare | 玄品 | 具备明确修行价值，通常要靠门路、声望或更高市场层级才能稳定获得。 |
| epic | 地品 | 稀有且强势，通常来自高阶拍卖、宗门库存或危险秘境。 |
| legendary | 天品 | 顶级传承与宝物，不会在普通流通链路里自然出现。 |

补充约束：

- 功法学习会消耗秘籍物品，且不可重复学习。
- 心法只允许同时启用一门；术法只要学会即可在战斗中使用，不限制已学数量。
- 熟练度按绝对值累计，每门功法满值不同，前端统一按百分比展示。
- 功法升阶后会得到一次可誊写秘籍的机会；达到圆熟后也可继续累积誊写次数。

## 9. 维护说明

- 任何物品、装备、秘籍、已学功法、悟道配方的改动，都必须同步更新本文件。
- 新增秘籍时，必须同时补“秘籍物品表”和“功法定义总表”；若为悟道专属，还必须补“悟道配方”。
- 发现专属秘籍默认不进入普通掉落、普通市场和普通拍卖池，除非明确改规则。
